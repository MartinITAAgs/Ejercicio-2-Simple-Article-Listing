## 📌 Descripción
Deberás desarrollar una página web con HTML y CSS que muestre un listado de artículos dados 
con un diseño flexible y responsivo. Empleando la etiqueta <article> para organizar y estructurar contenido.

## 📝 Reflexión
Este proyecto ha sido sencillo con el manejo de la etiqueta <article>, aunque se podría haber utilizado <div>, el primero 
resulto ser más efectivo a la hora de mantener la responsividad deseada y hacer entender que cada bloque sería más independiente.
